  This project is a simplified re-implementation of the standard C function printf, named ft_printf.
  Its purpose is to handle variadic arguments and produce formatted output to the standard output.
  Handled Specifiers:
 	%c	Single character.
	%s	String of characters
	%p	Pointer address 
	%d, %i	Signed decimal integer.
	%u	Unsigned decimal integer.
	%x	Hexadecimal lowercase.
	%X	Hexadecimal uppercase.
	%%	Literal percent sign.

ðŸš€ Instructions

The project is compiled using a Makefile to create the static library libftprintf.a. 
It requires a Libft to be present in a subdirectory named Libft.
